Trainings

Our goal is to bring high-quality, low-cost trainings to our community. The classes below range in price from $75 for a 1/2-day course to $300 for a 2-day course.

  • Web application hacking (2 days) - This class focuses on how to start finding and exploiting common web application vulnerabilities by hand and with tools.
  • Red teaming in the real world (2 days) - During this course we will look at the entire methodology of what it takes to execute a successful red team engagement.
  • Effective security leader training: A business-driven approach (1 day) - Being an effective security leader is a challenging prospect. This training outlines a method of practice which produces professional value.
  • Deep dive into cloud IAM applied to pen testing and vendor trust (4 hrs) - We will briefly compare IAM in Azure, Google, and AWS. Our deep dive will focus on AWS, but we will call out comparisons frequently.
  • If this then hack: intro to DIY cloud security automation w/ python (4 hrs) - Our workshop will discuss the core principles of what it takes to build your own automation tools for cloud security.
