More about our Platinum and Gold Sponsors

Platinum

Deepfactor

Deepfactor is a developer security platform that enables engineering teams to quickly discover and resolve security vulnerabilities and supply chain risks early in development and testing. With a single cost-effective solution, Deepfactor integrates runtime security, software composition analysis (SCA), software bill of materials (SBOM), and dynamic application security testing (DAST) to consolidate DevSecOps tooling. Unlike traditional AppSec tools, the Deepfactor runtime correlation technology automatically prioritizes and filters alerts for developers based on usage, providing critical context that streamlines remediation and reduces alert fatigue.

Gold

Traceable

Traceable is the industry's leading API security platform that protects your APIs by learning about all the APIs in your ecosystem, evaluating their risk posture to stop API attacks. Traceable does this by capturing EVERY API transaction, malicious or not, OVER TIME, in our API Data Lake, to get a full contextual understanding of your APIs and their normative baseline behavior. Traceable applies machine learning models on our API Data Lake to understand any anomalies and block API attacks, enabling organizations to be more secure and resilient.
